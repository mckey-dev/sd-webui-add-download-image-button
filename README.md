# sd-webui-add-download-image-button

**README 最終更新:** 2026-04-08

Stable Diffusion WebUI 向け拡張機能です。
生成画像を、ボタンまたはキーボードでブラウザダウンロードできます。

reForge のみ、動作確認済みです。

## 機能

- **ギャラリーのダウンロードボタン**: txt2img / img2img の出力ギャラリー直下のボタン行に ⭳ ボタンを追加します
- **モーダル（ライトボックス）のダウンロードボタン**: ギャラリー画像をクリックして開いた拡大表示内にも ⭳ を追加します
- **キーボードショートカット Shift+S**: 現在表示に応じた画像をダウンロードします（下記「ショートカットの対象」）

ダウンロードはブラウザの通常の保存ダイアログ／ダウンロードフォルダの挙動に従います。URL から取れる場合はファイル名を引き継ぎます。

## ショートカット（Shift+S）

| 状態 | ダウンロードする画像 |
|------|----------------------|
| モーダル（ライトボックス）が開いている | モーダルで表示中の画像 |
| 閉じていて txt2img タブが表示中 | txt2img ギャラリーの選択画像 |
| 閉じていて img2img タブが表示中 | img2img ギャラリーの選択画像 |

- プロンプトなど **入力欄・テキストエリア・セレクトにフォーカスがあるときは反応しません**（誤操作防止）
- **txt2img / img2img 以外のタブ**を前面にしているとき（設定・拡張など）は、モーダルが閉じていればショートカットはダウンロードしません
- 公式 UI のモーダル内「**s**」（小文字・Shift なし）は「ディスクに保存」と別操作です。本拡張の **Shift+S** とは競合しません

## ギャラリーで未選択のとき

サムネイルをまだ選んでいない場合でも、**最新（先頭）の画像**を対象にできるようになっています（実装および WebUI 側のギャラリー API に依存）。

## インストール

1. WebUI の `extensions` にこのフォルダを置くか、リポジトリをクローンします

```bash
cd extensions
git clone https://github.com/mckey-dev/sd-webui-add-download-image-button.git
```

2. WebUI を再起動するか、設定の「拡張機能」で一覧を更新します

## 使い方

1. txt2img または img2img で画像を生成する
2. ギャラリーで保存したい画像を選ぶ（または未選択のまま最新を対象にする）
3. **⭳ をクリック**するか、入力欄フォーカス以外で **Shift+S** を押す
4. ブラウザのダウンロード先に保存される

モーダル表示中は、モーダル内の ⭳ または Shift+S で **拡大表示中の画像**をダウンロードできます。
